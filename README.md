# telegrust-histo

#### Usage:

```sh
# install telegrust-histo with cargo, provided that you have the rust toolchain set up.
cargo install telegrust-histo

# cd to wherever your exported folder is, in my case:
cd ~/Downloads/Telegram\ Desktop/ChatExport_2021-01-15 

# run telegrust-histo 
telegrust-histo 
```
